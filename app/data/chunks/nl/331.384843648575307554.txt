# Mij ING ID - ING België - De ING ID op je ING-debetkaart

Leestijd: 1 min 09 oktober 2023 Je ING ID bestaat uit 10 cijfers en staat linksonder op de voorkant van je debetkaart Bancontact/Maestro en op de achterkant van je debetkaart Bancontact/Mastercard.

## Veelgestelde vragen

### Wat is een ING ID?

De ING ID is een unieke identificatiecode bestaande uit 10 cijfers waarmee ING jou als persoon kan identificeren.

### Is mijn ING ID hetzelfde als mijn rekeningnummer?

Neen, het gaat om 2 verschillende gegevens. Je hebt maar 1 ING ID, maar je kan meerdere ING-rekeningen hebben. Je vindt je rekeningnummer waaraan je debetkaart gekoppeld is boven je ING ID op de voorkant van je debetkaart Bancontact/Maestro en boven 'Account' op de achterkant van je debetkaart Bancontact/Mastercard.

### Is mijn ING ID hetzelfde als mijn kaartnummer?

Neen, het gaat om 2 verschillende gegevens. Je vindt je kaartnummer boven je naam op de voorkant van je debetkaart Bancontact/Maestro en boven 'Card' op de achterkant van je debetkaart Bancontact/Mastercard.

### Waarvoor moet ik mijn ING ID gebruiken?

Je ING ID moet je ingeven wanneer je ervoor kiest om je in te loggen in Home’Bank met behulp van je kaartlezer. Je ING ID gebruik je ook om een ING Banking profiel aan te maken. Dit is slechts eenmalig. Eens je ING Banking geactiveerd is, moet je je ING ID niet meer gebruiken.

### Wat moet ik doen als ik de ING ID niet meer kan lezen op mijn kaart?

Je kan eenvoudigweg een onleesbare of beschadigde vervangen via een aanvraag in Home’Bank of in de ING Banling-app.
