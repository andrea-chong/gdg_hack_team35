# Een kaart voor Bancontact, Maestro of Mastercard en contactloos betalen - ING België - ING-debetkaart

## Algemeen

### Hoe kan ik de online betalingen activeren/deactiveren?

Een vlucht boeken, concerttickets reserveren, kleren bestellen, een smartphone kopen ... Wellicht koop je ook regelmatig online? ING biedt je diverse oplossingen om elke onlinebetaling snel én veilig uit te voeren. Betaal standaard online met je krediet- of debetkaart. Maar je kan deze functie simpelweg deactiveren via onze digitale kanalen.
**Let op :** als je online betalingen deactiveert, worden ook alle periodieke betalingen van online abonnementen (bijvoorbeeld voor muziek, streamingdiensten, beveiligingssoftware … ) die verbonden zijn met je kaart geweigerd.
Om online betalingen activeren/deactiveren vanuit ING Banking app, volg de instructies hieronder.
1. Tik op 'Diensten' rechtsonder.
2. Selecteer 'Kaarten'.
3. Kies 'Debetkaarten' of 'Kredietkaarten'.
4. Tik op 'Online betalen'.
Ik activeer/deactiveer "online betalingen".

### Is het gebruik van de ING-debetkaart veilig ?

**Ja!** En die veiligheid wordt gegarandeerd dankzij je pincode. Je ING debetkaart is strikt persoonlijk en kan namelijk enkel gebruikt worden met het persoonlijke identificatienummer (pincode) dat alleen jij kent. Vertel niemand je code en schrijf deze ook nergens op. Kies voor een code die voor jou makkelijk te onthouden is, maar voor een ander moeilijk te raden.
