# ING Save Up voor minderjarigen – Spaar je wisselgeld als kind - ING België - ING Save Up voor jongeren onder de 18

## Help je kind eenvoudig en automatisch sparen!

Je bent nooit te jong om goede spaargewoonten aan te leren. Een beetje sparen bij elke in aanmerking komende betaling is een leuke manier om ongemerkt geld opzij te zetten. Kijk hoe kleine bedragen snel optellen!

## Wat is ING Save Up?

ING Save Up is een gratis dienst die je helpt sparen door bepaalde betalingen van je zichtrekening naar boven af te ronden op het dichtstbijzijnde hele eurobedrag. Aan het einde van elke dag wordt het totaal van deze afgeronde bedragen overgeschreven naar je spaarrekening.

#### Wat zijn de voorwaarden om deze dienst te gebruiken?

Om ING Save Up te gebruiken moet je:
* minstens 10 jaar oud zijn
* zowel een zichtrekening als een in aanmerking komende spaarrekening bij ING hebbe
Hoe activeer je ING Save Up voor je kind? Als je je kind volledige rechten(*Bij het openen van een spaarrekening voor je kind kun je ervoor kiezen om hem/haar het recht te geven het geld zelf te beheren vanaf 10 jaar. Dit kan je later aanpassen in je kantoor.) hebt gegeven op beide rekeningen, kan hij/zij de dienst zelf activeren via de ING Banking app. Anders kun jij als wettelijke vertegenwoordiger ING Save Up activeren voor je kind via je eigen app.
