# Consulter et modifier mes données personnelles - ING Belgique

## Questions fréquentes

### Comment mettre à jour ma carte d'identité ?

Mettez à jour votre carte d'identité dans un point CASH.
Comment ça marche ?
1. Insérez votre carte de débit dans le distributeur Cash By Bancontact (Batopin).
2. Appuyez sur « Modifier mes données personnelles ».
3. Reprenez votre carte de débit.
4. Insérez votre carte d'identité dans le distributeur Cash By Bancontact (Batopin).
Vous pouvez également mettre à jour votre carte d'identité (eID) au Kiosk dans une agence ING ou si vous disposez d’un lecteur de carte eID, vous pouvez aussi partager vos données mises à jour à distance avec ING via ce lien .
