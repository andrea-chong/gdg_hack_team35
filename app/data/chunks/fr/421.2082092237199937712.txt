# Vous n’avez pas reçu votre carte ? - ING Belgique

## Où et quand recevez-vous une nouvelle carte et un nouveau code PIN ?

### Pour info

ING offre désormais une carte de débit qui utilise le réseau Mastercard, plus étendu que le réseau Maestro. Toutes les cartes de débit existantes seront progressivement et automatiquement remplacées par la nouvelle carte de débit ING en 2024 et 2025. Vous recevrez la carte de débit Mastercard **dans un délai de 12 jours après réception du sms confirmant son envoi** **par la poste** . **Ne nous contactez pas avant la fin de ce délai.** Vous trouverez plus d’informations ici:
La carte de débit ING, avec Bancontact/Maestro // Bancontact/Mastercard et paiement sans contact.
* ### Après remplacement d’une carte endommagée, cassée ou illisible
Nous vous envoyons une nouvelle carte par courrier dans les 3-5 jours ouvrables. Votre code PIN reste le même que votre ancien.
Plus d'infos sur le remplacement de votre carte défectueuse
* ### Après remplacement suite à une perte ou à un vol
Lorsque vous aurez bloqué votre carte ING via l’application ING Banking, Home’Bank ou en appelant Card Stop, une nouvelle carte vous sera envoyée automatiquement par courrier dans un délai de 3 à 5 jours ouvrables. Cette nouvelle carte ING coûte 9,99 euros. Suivez les instructions fournies avec votre carte pour demander votre nouveau code PIN. Vous n’avez pas besoin de vous rendre dans une agence ING.
Plus d'infos sur une carte perdue ou volée ?
* ### Renouvellement avant l'échéance
Votre nouvelle carte vous sera envoyée au plus tard deux mois avant la date d’expiration. Votre code PIN reste le même que votre précédent.
* ### Une carte pour un nouveau compte à vue ou un nouveau titulaire
Vous devriez recevoir votre nouvelle carte par courrier dans les 3 à 5 jours ouvrables suivant cette ouverture. Suivez les instructions fournies avec votre carte pour demander votre nouveau code PIN.
