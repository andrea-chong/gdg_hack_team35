# Pourquoi ma carte de crédit ne fonctionne pas ? - ING Belgique - Votre carte de crédit ne fonctionne pas ou est bloquée ?

## Vous avez un problème avec votre code PIN ?

### Votre lecteur de carte n’accepte pas votre nouveau code PIN

Si vous utilisez une carte que vous venez de recevoir, activez-là avant de l’utiliser pour vos paiements via le lecteur de carte.
Utilisez votre nouvelle carte **pour la première fois** en effectuant une opération **avec le code PIN** : un achat chez un commerçant, un retrait d’argent (attention, les retraits sont généralement accompagnés de frais) ou une modification de votre code PIN à un distributeur automatique.
Votre carte de crédit ING **est alors activée** ! L’ancienne carte remplacée est automatiquement désactivée dès l'activation de la nouvelle.
Nous vous recommandons toutefois de la détruire en la coupant en deux au niveau de la puce.
**Attention** , les opérations sans contact ne permettent pas d'activer la carte.
