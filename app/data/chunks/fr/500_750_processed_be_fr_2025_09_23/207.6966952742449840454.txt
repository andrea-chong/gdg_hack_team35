# Activez votre nouvelle carte de crédit ING - ING Belgique

Temps de lecture: 5 min 16 novembre 2023 Suivez les étapes affichées sur cette page pour recevoir et activer votre carte de crédit

## Vous attendez votre nouvelle carte de crédit ING ?

Vous avez commandé une nouvelle carte de crédit ? Vous avez demandé le remplacement de votre carte car elle était illisible ? Ou après qu’elle ait été perdue ou volée?
Suivez simplement les étapes suivantes et votre carte de crédit sera prête à être utilisée.

## 1. Vous êtes informé(e) de l’arrivée de votre nouvelle carte de crédit par SMS

Dès que votre carte est envoyée, vous recevez un **SMS** sur le numéro de GSM enregistré dans nos fichiers.

## 2. Vous recevez votre carte de crédit par la Poste

**3 à 5 jours** après le SMS, vous recevez la carte de credit à l’adresse qui est enregistrée dans nos fichiers. Vous pouvez vérifier cette adresse dans l’app ING Banking (Services > Données personnelles) ou dans Home'Bank ("Mes paramètres").

## 3. Consultez ou demandez votre code PIN

Vous avez reçu une **carte de remplacement** pour votre carte de crédit expirée ou inutilisable ? **Vous gardez simplement le même code PIN.**
Vous avez demandé une **nouvelle carte** ou la vôtre a été **perdue ou volée** ? Vous pouvez alors
* soit **consulter le nouveau code PIN** dans l'app ING Banking : Services > Cartes > Code Pin > Consulter
* soit **demander que le nouveau code PIN vous soit envoyé par sms** . Suivez les instructions fournies dans le courrier envoyé avec votre carte et veillez à bien utiliser le numéro de GSM que vous nous avez transmis. Vous pouvez vérifier ce numéro dans l'app ING Banking: Services > Données personnelles ou dans Home'Bank ("Mes paramètres").
Vous avez **12 jours** pour demander votre code PIN. Passé ce délai ou si vous ne possédez pas de GSM, le nouveau code PIN sera envoyé par courrier à la même adresse que votre carte de crédit.
Une fois votre code PIN reçu, vous pouvez le modifier via n'importe quel point CASH Bancontact .
