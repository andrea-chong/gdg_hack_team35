# Le compte d'épargne pour les jeunes - ING Belgique - Le compte épargne pour les jeunes

## Questions fréquemment posées

### Dois-je posséder un ING Compte à vue pour pouvoir ouvrir un ING Compte d’épargne pour mon enfant ?

Si vous n’êtes pas encore client ING, vous devez ouvrir votre compte d’épargne pour votre enfant via l’une de nos agences. Si vous possédez déjà un compte ING (Compte à vue, Compte d’épargne ou Compte d’investissement), vous pouvez ouvrir ce compte aussi bien via nos canaux digitaux qu’en agence.
