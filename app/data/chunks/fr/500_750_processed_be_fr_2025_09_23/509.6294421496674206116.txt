# Activez votre nouvelle carte - ING Belgique - Activez votre nouvelle carte de débit ING

## 3. Consultez ou demandez votre code PIN

Vous avez reçu une **carte supplémentaire** ou une **carte de remplacement** pour votre carte expirée ou inutilisable ? Vous gardez simplement le même code PIN.
Vous avez demandé une **nouvelle carte** ou la vôtre a été **perdue ou volée** ? Vous pouvez vérifier votre code PIN dans notre application (via le scan du QR code sur la lettre ou y allez vous-même dans l'app - services>cartes> regarder le pin) ou **v** **ous pouvez demander un nouveau code PIN par SMS** ou **contacter ING par téléphone, ou vous rendre en agence.** PIN via SMS: suivez les instructions fournies dans le courrier envoyé avec votre carte pour demander un nouveau code PIN par SMS. Veillez à bien utiliser le numéro de téléphone mobile que vous nous avez transmis. Vous pouvez vérifier ce numéro dans votre application bancaire ou via Home'Bank (dans "Mes paramètres"). Vous avez **12 jours** pour demander votre code secret. Pour des raisons de sécurité, le message contenant votre code secret sera automatiquement supprimé 2 jours après l'envoi. Si vous n'avez pas consulté ou demandé de nouveau code secret dans les **12 jours** ou si vous ne possédez pas de smartphone, nous vous envoyons automatiquement votre code secret par courrier à la même adresse que votre carte. Une fois votre code reçu, vous pouvez le modifier via n'importe quel distributeur automatique (ING/Cash point/ autre banque) .
Vous avez contacté ING: vous pourrez choisir votre code Pin sur un Cash point or distributeur automatique ING dans un délai de 2 heures.
