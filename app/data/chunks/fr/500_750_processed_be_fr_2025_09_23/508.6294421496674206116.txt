# Activez votre nouvelle carte - ING Belgique - Activez votre nouvelle carte de débit ING

Temps de lecture: 5 min 28 décembre 2023 Suivez les étapes décrites sur cette page pour recevoir et activer votre carte

## Vous attendez une nouvelle carte de débit ING ?

Vous avez demandé une nouvelle carte ou un nouveau compte à vue auprès d'ING ? Ou vous attendez une carte de remplacement après que la vôtre ait été volée, perdue ou a expiré ?
Pour la recevoir et demander, si nécessaire, un nouveau code secret, il vous suffit de suivre les étapes décrites sur cette page et votre carte sera prête à être utilisée.

## 1. Scannez le QR code sur la lettre ou un SMS vous arrivera bientôt

Dès que votre carte est envoyée, vous recevez un **SMS** sur le numéro de téléphone mobile repris dans votre dossier. Ou si vous avez installé l'application ING bancaire avec itsme, vous pouvez scanner le code QR.

## 2. Vous recevez votre carte par courrier

Vous devriez recevoir votre nouvelle carte de débit par courrier dans les **3 à 5 jours ouvrables** qui suivent la réception de ce SMS. Si vous êtes déjà client, nous utilisons **l'adresse** que vous nous avez transmise. Vous pouvez vérifier cette adresse via, votre application bancaire ou dans Home'Bank ( « Services » > « Données personnelles »). Si vous êtes un nouveau client, nous utilisons l'adresse fournie lors de votre demande de carte de débit.
