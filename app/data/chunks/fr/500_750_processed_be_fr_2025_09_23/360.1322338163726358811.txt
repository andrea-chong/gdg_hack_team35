# Comment consulter votre code PIN et débloquer votre carte ? - ING Belgique - Consultez le code PIN ou débloquez votre carte de débit ou crédit

Temps de lecture: 4 min 20 novembre 2023 Une nouvelle carte de débit ? Ou avez-vous oublié votre code PIN ? Avez-vous bloqué votre carte après avoir introduit un code erroné à trois reprises ? Consultez votre code PIN ou débloquez votre carte de débit ou crédit en utilisant l'app ING Banking.

## Comment consulter ou débloquer le code PIN dans l'app ING Banking?

Vous pouvez facilement consulter et/ou débloquer le code PIN de votre carte de débit ou crédit dans votre app ING Banking en moins d'une minute. Découvrir dans votre app .
1. 1

#### Allez dans « Services »

Ouvrir l'app ING Banking et allez dans « Services » en bas à droite de l'application.
2. 2

#### Choisissez l'option « Cartes » et ensuite « Code PIN »

En sélectionnant le service « Code PIN », vous aurez la possibilité de consulter le code PIN ou de débloquer une carte.
3. 3

#### Choisissez la carte dont vous voulez voir le code pin

Votre code PIN s'affiche chiffre par chiffre.

### Ou regardez la vidéo ci-dessous avec des instructions pas à pas.

Points d’attention lors de la consultation de votre code PIN :
* Mémorisez votre code PIN **sans le noter**
* Vous recevrez un mail (à l’adresse enregistrée dans notre système) vous informant que vous avez consulté votre code PIN
* Vous ne recevrez **jamais de SMS** de notre part pour demander le code PIN, ne donner pas suite à ces messages
