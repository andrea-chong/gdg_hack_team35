# Extraits de compte - Foire aux questions - ING Belgique - Extraits de compte Questions Fréquemment posées

### Comment modifier le mode de réception d'extraits de compte?

Vous avez 2 méthodes pour en modifier le mode de réception :
* Home'Bank ou Business'Bank > Gérer le quotidien > Mes comptes à vue > Extraits de compte.
* Home'Bank ou Business'Bank > Gérer le quotidien > Mes comptes à vue puis cliquer sur "Actions" à côté du compte.
Sélectionnez ensuite "Modifier le mode de réception des extraits".
Vous pouvez modifier ce mode de réception pour plusieurs comptes à la fois : sélectionnez les comptes désirés > Actions sur sélection > Modifier le mode de réception des extraits.
Attention : si vous souhaitez repasser aux extraits papier, mais que vous n'avez pas encore demandé tous vos extraits, ces derniers feront l'objet d'une demande automatique et d'une dernière remise par voie électronique.

### Est-ce qu'un changement du mode de réception d'extraits impacte les autres qui y ont accès?

Oui. Dans le cas où plusieurs personnes ont accès à un compte donné : si une d’entre elles change le mode de réception via Home'Bank ou Business'Bank – passe d’envoi postal à envoi électronique – cela s’applique à tout le monde. Même si c’est dans le cadre d’un second envoi.

### Est-il possible de demander d'extraits électroniques pour tous les comptes?

Oui, bien sûr. Néanmoins, vous devez introduire une demande séparée pour chaque compte.
Vous contrôlez les paramètres de vos extraits de compte dans Home'Bank ou Business'Bank .

### Dois-je imprimer mes extraits de compte électroniques?

Non, ce n'est pas une obligation. L'un des atouts majeurs des extraits de compte électroniques est que vous pouvez les sauvegarder sur votre disque dur. Ainsi, vous épargnez du papier et contribuez à la protection de l'environnement. Bien évidemment, si vous avez vraiment besoin d'une copie papier, libre à vous d'imprimer le ou les extraits nécessaires.
