## 4. Utilisez votre code PIN pour activer votre carte

Votre carte est automatiquement activée **la première fois** que vous effectuez une opération qui nécessite d'introduire votre **code PIN** , par exemple :
* changer votre code PIN via un distributeur automatique - Cash point ou ING ou autre distributeur(recommandé)
* effectuer un paiement en magasin
* retirer de l'argent via un distributeur (ING ou autre)
Si votre nouvelle carte vous a été envoyée dans le but de remplacer une ancienne carte, celle-ci sera automatiquement désactivée dès l'activation de la nouvelle. Néanmoins, nous vous recommandons de détruire votre ancienne carte en coupant en deux avec des ciseaux tant la carte que la puce qu'elle contient.

## Questions fréquentes

### Que faire si je ne reçois pas la carte de débit ?

Si vous ne recevez pas votre carte de débit, vérifiez l’adresse connue chez nous et corrigez-la si nécessaire via l’App ING Banking ou via Home’Bank.
Si vos données personnelles sont à jour, vous pouvez commander une nouvelle carte de débit en ligne.
Commander une nouvelle carte de débit

### Que faire si je n’ai pas demandé le code PIN via SMS dans le délai des 12 jours ?

Vous ne devez rien faire, nous vous l’envoyons par courrier.

### Que faire si je ne me souviens pas de mon code PIN ?

Vous pouvez le consulter facilement via l’App ING Banking.

### Que faire de mon ancienne carte ?

Nous vous recommandons de détruire votre ancienne carte en la coupant avec des ciseaux à travers la puce de la carte.

### Que faire si ma carte de débit est arrivée en agence ?

Vous pouvez récupérer votre carte de débit dans votre agence entre 9h et 12h.
