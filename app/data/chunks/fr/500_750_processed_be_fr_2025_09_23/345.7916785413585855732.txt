# ING Save Up pour les mineurs – Épargnez votre petite monnaie dès le plus jeune âge - ING Belgique - ING Save Up pour les moins de 18 ans

## Aidez votre enfant à épargner facilement et automatiquement !

Il n’est jamais trop tôt pour adopter de bonnes habitudes d’épargne. Mettre un peu de côté à chaque paiement éligible est une manière ludique d’épargner sans même s’en rendre compte. Observez comment les petits montants s’additionnent !

## Qu’est-ce qu'ING Save Up ?

ING Save Up est un service gratuit qui vous aide à épargner en arrondissant certains paiements effectués depuis votre compte à vue à l’euro supérieur. En fin de journée, le total de ces arrondis est transféré vers votre compte d’épargne.

#### Quelles sont les conditions pour utiliser ce service ?

Pour utiliser ING Save Up, il faut :
* avoir au moins 10 ans
* détenir un compte à vue et un compte d’épargne éligible chez ING.
Comment activer ING Save Up pour votre enfant Si vous avez donné à votre enfant les droits complets(*Lors de l’ouverture d’un compte d’épargne pour votre enfant, vous pouvez lui accorder le droit de gérer son argent s’il a plus de 10 ans. Ce paramètre peut être modifié en agence.) sur les deux comptes, il peut activer le service lui-même via l’app ING Banking. Sinon, vous pouvez, en tant que représentant légal, activer ING Save Up pour votre enfant via votre propre app.
