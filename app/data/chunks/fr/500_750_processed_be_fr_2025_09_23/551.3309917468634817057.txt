# Consulter et modifier mes données personnelles - ING Belgique

## Questions fréquentes

### Comment puis-je modifier les données personnelles de mon/mes enfant(s) ?

Vous pouvez adapter les données personnelles de vos enfants via leur profil dans l’app ING Banking ou Home’Bank.
1. Connectez-vous à l’app ou au Home’Bank de votre enfant.
2. Rendez-vous dans « Données personnelles ». Vous verrez un aperçu de toutes les données personnelles de votre enfant. Des informations sont manquantes ou obsolètes ? Vous pouvez les adapter immédiatement.
Vous avez déménagé ? Dans ce cas, vous pouvez aussi indiquer l’adresse de vos enfants en adaptant votre propre adresse légale.
Attention : c’est uniquement possible pour les enfants connus chez ING.
Vous aimeriez apprendre à votre enfant à gérer son argent ? Découvrez le compte jeunes ING Go to 18 .

### Je n’ai pas itsme®, comment puis-je mettre à jour mes données personnelles ?

Si vous n’avez pas itsme®, vous pouvez facilement adapter vos données personnelles manuellement dans l’app ING Banking ou dans Home’Bank.
Sinon, vous pouvez vous rendre dans un point CASH muni de votre carte de débit ING et de votre carte d’identité : la synchronisation de vos données d’identité se fait alors instantanément.
