# Compte épargne à long terme gratuit - ING Belgique - Le compte d'épargne classique

## Questions fréquemment posées

### Comment ajouter un compte de référence externe ?

Via le formulaire en ligne : sélectionnez le compte d’épargne réglementé auquel vous souhaitez lier un compte de référence externe, indiquez votre compte de référence externe détenu dans une autre banque belge et effectuez un virement de minimum 0,01€ de ce compte externe vers votre compte d’épargne reglementé ING.
