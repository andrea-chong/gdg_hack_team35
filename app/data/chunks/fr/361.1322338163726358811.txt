# Comment consulter votre code PIN et débloquer votre carte ? - ING Belgique - Consultez le code PIN ou débloquez votre carte de débit ou crédit

## Questions fréquemment posées

### Dois-je faire autre chose après avoir consulté le code PIN ou débloqué ma carte ?

Oui, pour une nouvelle carte- voir question au -dessus. Non, une fois que vous avez consulté votre code PIN ou débloqué votre carte, vous n'avez rien d'autre à faire pour profiter des avantages de votre carte.

### Puis-je également consulter le code PIN d'une nouvelle carte ?

Bien sûr ! Toutefois, pour utiliser votre carte, vous devez l'activer en effectuant une transaction avec le code PIN reçu, par exemple un achat dans un magasin avec votre carte (pas en ligne) ou un retrait d'argent à un distributeur automatique de billets.

### Puis-je également consulter le code PIN ou débloquer ma carte de crédit ING ?

Oui, vous pouvez également consulter le code PIN ou débloquer votre carte de crédit ING via la même option. Sur l'écran de sélection de la carte, en sélectionnant la carte de crédit.

### Puis-je également consulter le code PIN ou débloquer la carte d'un enfant mineur ?

L'enfant mineur peut consulter le code PIN ou débloquer une carte depuis son ING Banking app :
1. Rendez-vous dans le menu « Services », situé en bas à droite de l'écran principal
2. Choisissez « Cartes »
3. Sélectionnez « Voir codes PIN de vos cartes »
L'application ING Banking est disponible à partir de 10 ans.

### Puis-je également débloquer ma carte de débit ou crédit si je l'ai bloquée dans le lecteur de cartes ?

Non, vous devez l'activer en effectuant une transaction avec un code PIN, par exemple un achat dans un magasin avec votre carte (pas en ligne) ou un retrait d'argent à un distributeur automatique.

### Puis-je modifier le code PIN de ma carte ?

Allez à un point Cash ou distributeur automatique (ING ou autre banque) pour modifier le code PIN de votre carte.
